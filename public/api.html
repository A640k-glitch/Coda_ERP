Enterprise API Integrations

Integrate Coda's real-time banking, compliance, and messaging modules. To use these endpoints, supply the respective API keys in your backend environment.

POST Verify CAC Registration

/api/v1/integrations/cac/verify

Query CAC registry to validate business numbers (RC/BN).

Field Type Description
rcNumber string CAC number (e.g. RC1234567)
companyName string Optional. Registered name for cross-check

POST Sync Bank Statement (Mono)

/api/v1/integrations/bank/sync

Fetch real-time transaction feeds from connected Nigerian bank accounts for auto-reconciliation.

Field Type Description
accountId string Mono connected account ID
period string Date range e.g., last30days

POST Automated Tax Filing (FIRS)

/api/v1/integrations/tax/file

Remit VAT/WHT directly to TaxPromax.

Field Type Description
tin string FIRS Tax Identification Number
period string Tax period e.g. 2026-06
amount number Total tax amount (NGN)

POST Send SMS via Termii

/api/v1/integrations/notifications/sms

Send high-deliverability SMS for OTPs and payment reminders.

Field Type Description
to string Recipient Phone (e.g. 23480...)
message string SMS Content
