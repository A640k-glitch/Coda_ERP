API Reference

Integrate Coda directly into your operations using our RESTful JSON endpoints.

POST Business Onboarding

/api/v1/business/onboard

Onboard a new business branch or subsidiary to the ERP platform.

Field Type Description
businessName string Registered name of the company
registrationNumber string CAC number (e.g. RC1234567)
contactEmail string Primary contact email address

POST Record Ledger Entry

/api/v1/accounting/transactions

Direct post a double-entry transaction to accounts ledger.

Field Type Description
date string Optional ISO-8601 date of transaction
description string Transaction description/memo
lines array Array of objects: { account, debit, credit }

POST Calculate Tax Liabilities

/api/v1/tax/calculate

Programmatic payroll tax computation using 2026 guidelines.

Field Type Description
amount number Naira amount base
taxType string vat (7.5%), wht (10%), paye

POST Create Product

/api/v1/inventory/products

Add a new product to your inventory catalog.

Field Type Description
name string Product name
sku string Stock Keeping Unit identifier
cost_price number Base cost price (NGN)
sell_price number Target selling price (NGN)
Integration Guide & Examples

To integrate with the Coda API, generate an API Key from your Admin Dashboard and include it in the x-api-key header of your requests. All API endpoints accept and return JSON.

cURL Example
curl -X POST https://api.coda.ng/api/v1/tax/calculate \
-H "Content-Type: application/json" \
-H "x-api-key: YOUR_API_KEY_HERE" \
-d '{"amount": 100000, "taxType": "vat"}'
Node.js / Fetch Example
const
response =
await

('https://api.coda.ng/api/v1/tax/calculate', {
